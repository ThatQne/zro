what's actually in it
rendering
Per-tab WebView2 renderers. UI overlays composite over the page via a live GDI region — panels never shift the viewport, rounded corners on every site.
memory
Inactive tabs freeze after a minute, then fully sleep. RAM stays flat at hundreds of tabs; the whole app freezes when the machine goes idle.
shields
Brave's native adblock engine compiled in. Blocks ads + trackers, upgrades HTTP→HTTPS, strips tracking params. Pauses on tabs playing audio.
ai agent
Ollama / OpenAI-compatible / local CLI. Tool loop over read_page, click, fill, navigate, history — with a persistent embeddings memory.
privacy
Incognito with a Windows Hello lock, a built-in cookie editor, a local password viewer. Nothing leaves the machine.
updates
Signed builds delivered in-app from GitHub Releases. One click, no reinstall — verified against a minisign key before install.
footprint
4.4 MB installer, ~16 MB on disk. Renders through the system WebView2 runtime — no bundled Chromium, not a 200 MB Electron download.
stack
Rust · Tauri v2 · React · TypeScript. MIT licensed, developed in the open.
